# video-collection-player
网络视频采集播放器，自行提供可访问的rss视频源
# 说明
旨在flutter学习交流，本软件不提供任何视频信息，所有视频均是使用者自行获取，与作者无关，🚫🚘，不要提任何与技术无关的话题

本软件只是本人在休息时做的一个玩具，第一次玩flutter，欢迎批评指正，大家一起学习flutter
